# Cucumber BDD Test Project

This project serves as a proof of concept for Behavior-Driven Development (BDD) using Cucumber in JavaScript. It demonstrates how to write and execute Cucumber scenarios for testing an API that retrieves country information.

## Overview

The purpose of this project is to showcase how BDD principles can be applied using Cucumber to improve collaboration between development, testing, and other stakeholders. The focus is on testing an API endpoint that retrieves country information using Axios for HTTP requests.

## Features

- **Cucumber Scenarios:** Define BDD scenarios using Gherkin syntax.
- **Axios:** Make HTTP requests to the API endpoint.
